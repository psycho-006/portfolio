Timeline

The timeline shows how I build website

  • 1st

    Wireframe

    Wireframing is important step in any screen design process. It primarily allows me to define the information hierarchy of my design, making it easier for me to plan the layout according to how I want my user to process the information.

  • 2nd

    Coding

    Writing the necessary source code to create a website. At the very least, it refers to writing the HTML, CSS and JavaScript. JavaScript programing allows to turn a simple, static HTML page into a dynamic masterpriece. It allows others to interact with website and use the application on any computer with internet access.

  • 3rd

    Testing & Debugging

    Testing is meant to find issue in the code and debugging is the process of finding and removing a specific bug from the code. I test the code in all modern browser and phone trying to find issue. If any issue found I fix it.

  • 4th

    Optimize

    Website optimization is the process of systematically improving the performance of a website. The process for optimizing website follows the same principles as conversion rate optimization. I optimize website according to Google PageSpeed Insights recommendation.
